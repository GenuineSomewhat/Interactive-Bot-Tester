HOW IT WORKS:

1. You type: !planegame
2. Bot loads aircraft data from Wikipedia
3. Plane game thread calls: send_message_with_image(msg, url)
4. OUR PATCH intercepts this call
5. Image URL stored in message_responses list
6. GUI reads message_responses
7. GUI downloads image from Wikimedia Commons
8. GUI displays image in the chat window

RESULT: Image shows up in your GUI, NOT sent to GroupMe!
